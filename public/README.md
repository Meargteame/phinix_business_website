# Project Screenshots

Please save your project screenshots in this folder with the following names:

1. **yaricho-senior-home-care.png** - Screenshot of the Yaricho Senior Home Care website
2. **torra-parking.png** - Screenshot of the Torra Parking website  
3. **ecommerce-platform.png** - Screenshot of the ShopHub e-commerce website

These images will be displayed in the "Selected Works" section of your Phoenix Enterprise landing page.

## Image Requirements
- Format: PNG or JPG
- Recommended size: 1200x800px or similar aspect ratio (4:3)
- Quality: High resolution for best display

The images are already referenced in the code and will display automatically once placed in this folder.
